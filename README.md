CAIA Stewardship
================

This repository contains non-normative materials intended to support
understanding, interoperability, and adoption of the CAIA specification.

Examples of content that may appear here include:
- Interoperability guidance
- Draft certification or badging concepts
- Reference architectures
- Best practices and implementation notes
- Example code and test materials

This repository does NOT contain the CAIA specification.

Normative authority over the CAIA specification resides exclusively with
the CAIA Maintainers Group (CMG) and the caia-spec organization.
